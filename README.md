# ВЕБ-ПРОЕКТ "*MEDEBOL*"

## Суть проекта:

Проект был создан в образовательных целях. Данный проект пойдет в портфолио как кейс. Что бы показать мои скиллы в веб-технологиях. Проект не нарушает авторских прав, а картинки и текст были сгенерированы при помощи искусственного интеллекта.

## Описание проекта:

Веб-сайт интернет магазин, продажа мебели. Как аналог IKEA. Всего количество товаров 15 штук. Разный тип товаров от диванов до стол. При нажатии на кнопку можно увидеть полное описание товара а так же его цену. Присутствует корзина, туда можно кидать желаемые товары. Если вы кинули товары в корзину, кнопка оформить заказ становится доступной. Но что бы зказать товары, нужно ввести информацию, такие как: Email и имя. Присутсвует валидация формы. Если вся информая верна, заказ был оформлен успешно. Так же, есть 3 способа найти нужный товар. 1-ый самый простой: проскролить страницу пока не найдешь. 2-ой, наверху страницы есть кнопка каталог, при наведении которой он предлагает вид товара, такие как диван и т. п., при нажатии которого он вас перекидывает в нужный раздел с товарами. И 3-ий способ, на странице присутствует фильтрация, а конкретно поле ввода. При написании нужного товара, предлагает товар по описанию. На старнице можно менят тему со светлого на темную тему. Так же на странице есть слайдер с треями интересными предложениями. В проекте есть 2 страница, называется "О нас". Там рассказывается про бренд и что они делают. Красиво оформлено и напоминает коммерческий проект.

## Технологии которые используются в проекте:

* HTML
* CSS
* JS

*В проекте используются normalize.css, Roboto шрифт, и кастомный шрифт MEDEBOL.*

## Методы написания кода в проекте:

### HTML

HTML код структурирован. Используются семантические теги такие как section, dialog, main. Используются в основном классы для обращенния к тегам, классы понятны наименованны. Для картинок используются атрибуты width и height, что бы уровень CLS был максимально близок к 0. К этим же картинкам присутствует атрибут: loading="lazy", что бы скорость загрузки на устройствах повысилась. В конце HTML кода написан код в основном для декорации. В head используются мета теги для лучшего рейтинга сайта.

### CSS

В CSS используются современные методы написания кода, а так же сам код. Используется метод написания кода Mobile-First, в конце CSS файла код для медиа запросов. Используются классы к классам, что бы применить доп стили, которые будут вызванны интерактивным JS.

### JS

В JS коде используется большое количество комментариев, для лучшего понимания кода. Используется большое количество глобальных перемен особенно константы для DOM элементов, присутствует делегирование событий для выбора товара, для товаров в списке. Логика с корзиной. Функция для сокращенния текста. Логика изменения темы сайта. Логика фильтрации в инпуте. Вызов пару анимаций при помощи JS, такие как скролл и хедер отодвигается.

## Как запустить проект:

В папке проекта MEDEBOL, в корне папки MEDEBOL будет файл index.html, нажав на него он откроется. Его можно на любых браузеров. Помимо этого, есть второй способ, если у вас есть расширение live server, то в редакторе VC Code откройте папку MEDEBOL, откройте файл index.html, и правым кликом по редактору, появится окно, где будет кнопка "Открыть при помощи Live Server".
